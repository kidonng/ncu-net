## Usage

1. Setup `npm i`
2. Install a process manager (e.g. pm2 `npm i pm2 -g`)
3. [Config username, password & more](ncu-net.js#L1-L18).
4. Start a daemon (e.g. `pm2 start ncu-net.js`)
